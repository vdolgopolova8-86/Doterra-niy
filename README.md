# doTERRA Inventory

Личный веб-приложение для учёта коллекции эфирных масел doTERRA: наличие, остатки, минимальный запас и список покупок.

## Возможности

- **Учёт масел** — закрытые/открытые флаконы, объём, категории, заметки
- **Статистика** — наглядные карточки по коллекции
- **Список покупок** — автоматически при низком запасе + ручное добавление
- **Поиск и фильтры** — по названию, состоянию и категории
- **Экспорт/импорт** — резервная копия JSON, список покупок CSV
- **Локальное хранение** — данные сохраняются в `localStorage` браузера

## Быстрый старт

### Вариант 1: открыть файл

Откройте `index.html` в браузере двойным щелчком.

> Для корректной работы некоторых функций (импорт/экспорт) рекомендуется локальный сервер.

### Вариант 2: локальный сервер

```bash
npm install
npm start
```

После запуска откройте [http://localhost:3000](http://localhost:3000).

## Использование

1. На главной странице нажмите **«Открыть приложение»**
2. При первом запуске добавятся примеры масел (Лаванда, Лимон, On Guard)
3. Добавляйте свои масла через форму слева
4. Открывайте **«Список покупок»** для планирования заказа
5. Делайте **экспорт JSON** для резервной копии
